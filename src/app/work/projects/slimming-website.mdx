## Overview

A web platform built for a weight loss program business, allowing them to manage clients, track progress, and deliver program content. Built with Next.js for the frontend and Supabase as the backend-as-a-service layer.

## Key Features

- **Client Management**: Dashboard for managing client registrations, progress tracking, and program assignments.
- **Content Delivery**: Structured pages for delivering program materials, meal plans, and workout guides.
- **Supabase Integration**: Used Supabase for authentication, database, and real-time data features — allowing rapid backend setup without managing infrastructure.
- **Responsive Design**: Mobile-first layout ensuring a consistent experience across devices.

## Technologies Used

- **Next.js**: React-based frontend with server-side rendering.
- **Supabase**: Backend-as-a-service for auth, database, and storage.
- **Tailwind CSS**: Utility-first styling for fast, consistent UI development.

## Challenges and Learnings

Balancing a clean user experience with the complexity of managing client data required careful information architecture. Supabase's row-level security made it straightforward to enforce data access rules without building a custom auth system.

## Outcome

Delivered a working platform that the business uses to onboard and manage their weight loss program clients, reducing manual coordination and improving their service delivery.

Live at [lumina-slim.vercel.app](https://lumina-slim.vercel.app/).
