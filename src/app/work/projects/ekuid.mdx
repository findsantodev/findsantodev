## Overview

Ekuid is a securities crowdfunding platform that connects investors with impactful projects in society. I joined as the sole Mobile Engineer and was responsible for building the app from the ground up — from architecture decisions to App Store and Google Play distribution.

## Key Features

- **Flutter App from Scratch**: Designed and built the complete mobile application using Flutter and GraphQL, delivering a smooth cross-platform experience on both iOS and Android.
- **Clean Architecture**: Structured the codebase using Clean Architecture principles, keeping business logic, data, and presentation layers clearly separated for long-term maintainability.
- **CI/CD with Codemagic**: Set up automated build and distribution pipelines using Codemagic, enabling reliable releases to both the App Store and Google Play with minimal manual effort.
- **Offline Mode**: Designed and implemented offline support, allowing users to browse and access key content without an active internet connection.
- **Project Management Flow**: Defined and maintained the team's project management workflow using Notion, improving cross-functional visibility and delivery consistency.

## Technologies Used

- **Flutter & Dart**: Cross-platform mobile development.
- **GraphQL**: Efficient, typed API communication.
- **Codemagic**: CI/CD pipeline for automated builds and store distribution.
- **Firebase**: Push notifications and analytics.
- **Clean Architecture**: Scalable project structure.

## Challenges and Learnings

One of the main challenges was designing an architecture that could scale as new features were added over a multi-year timeline. Clean Architecture made it possible to swap out data sources, add new use cases, and onboard contributors without breaking existing functionality. Setting up Codemagic early saved significant time across hundreds of releases.

## Outcome

The Ekuid app was successfully launched on the [App Store and Google Play](https://eku.id), serving real investors and project owners. The app remained stable and maintainable over 4 years of continuous development and iteration.
