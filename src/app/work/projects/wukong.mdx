## Overview

Wukong is a platform built from scratch, where I served as the sole Full Stack Engineer responsible for both frontend and backend architecture. The goal was to deliver a production-ready, scalable system with a solid engineering foundation.

## Key Features

- **Full Stack Architecture**: Designed and implemented the entire platform — frontend in Next.js and backend in Golang — ensuring clean separation of concerns and scalability.
- **API Design**: Built and structured RESTful APIs to support core platform features with consistent, documented endpoints.
- **State Management**: Set up frontend state management and data-fetching patterns optimized for performance and maintainability.
- **Deployment Pipeline**: Established CI/CD pipelines and deployment infrastructure to support continuous delivery.

## Technologies Used

- **Next.js**: Server-rendered React frontend for fast, SEO-friendly pages.
- **Golang**: High-performance backend service handling business logic and data management.
- **REST API**: Clean API contracts between frontend and backend.
- **CI/CD**: Automated deployment pipeline for reliable releases.

## Challenges and Learnings

Building the entire platform solo meant making architectural decisions that needed to hold up over time. Choosing Golang for the backend allowed for high concurrency and fast response times, while Next.js on the frontend enabled quick iteration on UI without sacrificing performance. Establishing consistent API contracts early was critical to keeping both sides in sync.

## Outcome

Successfully launched the Wukong platform at [wukong.co.id](https://wukong.co.id), delivering a stable, scalable product with a solid engineering foundation ready for future growth.
